Welcome to Analog Journal

Analog Journal is an application for managing analog films and photographs. It helps you track films, cameras, and details of each individual frame.

All data is stored locally on your device - you don't need an account or internet connection for basic features.

rocket_launch Quick Start

  1. 1 Add your cameras in the Cameras section
  2. 2 Create a new film on the home screen
  3. 3 Load the film into a camera in Camera Detail
  4. 4 After each shot, add a record in Film Detail
  5. 5 After shooting the film, mark it as developed

sync_alt Film States

photo_camera

Active Film

Film is loaded in a camera and ready for shooting. You can add frames. Shown with green border.

check_circle

Developed Film

Film has been shot and developed (archived). Cannot add more frames, but you can revert the status. Shown with blue border.

pending

Undeveloped Film

Film is waiting to be shot or developed. You can load it into a camera and add frames. Shown with gray border.

database Data Management

download Export Data

In Settings, click "Export Data" to download a JSON file with all data (films, frames, cameras). Keep this file as a backup.

upload Import Data

In Settings, click "Import Data" and select a JSON file from previous export. System checks for duplicates and imports only new data.

warning Warning: Data is stored on your device. Clearing browser/app data will delete app data. Export backups regularly!
